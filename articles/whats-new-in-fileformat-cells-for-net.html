Version 24.1.0 (Release Date: January 17, 2024)

New Row and Column Manipulation Features

In this latest release of FileFormat.Cells for .NET (Version X.X.X), we are excited to introduce powerful new features for manipulating rows and columns in Excel spreadsheets. These enhancements are designed to make your spreadsheet automation tasks even more versatile and efficient.

Key Updates:

  • InsertRow and InsertRows Methods: We have implemented new methods, InsertRow and InsertRows, allowing you to effortlessly insert rows within Excel worksheets programmatically.

  • HideRows and UnhideRows Methods: With the addition of HideRows and UnhideRows methods, you can now easily hide and unhide rows in your Excel worksheets, providing greater control over data presentation.

  • HideColumns and UnhideColumns Methods: Similarly, we have introduced HideColumns and UnhideColumns methods for efficient column management.

  • Enhanced Worksheet Class: The Worksheet class has been enhanced to support advanced row and column operations, further improving the functionality of the library.

  • Robust Error Handling: We have ensured robust error handling and input validation for all the newly added methods, making your code more reliable.

These updates significantly expand the capabilities of the FileFormat.Cells library, enabling you to manipulate rows and columns in Excel worksheets with ease and precision. Whether you're inserting, hiding, or unhiding rows and columns, FileFormat.Cells for .NET Version X.X.X has you covered.

Upgrade now to take advantage of these powerful new features and enhance your spreadsheet automation workflows!

Version 23.12.0 (Release Date: December 18, 2023)

Implementation of ValidationRule Support in Worksheet and Range Classes

We are excited to announce a significant enhancement to the Worksheet and Range classes in FileFormat.Cells. In this update, we have introduced comprehensive support for various types of data validations through the new ValidationRule class. This enhancement brings advanced data validation capabilities to your Excel automation tasks.

Key Highlights:

  • Extended ValidationRule Class: The ValidationRule class has been extended to support a diverse range of data validations, including List, WholeNumber, Decimal, TextLength, and CustomFormula types. You now have more options to enforce data integrity in your Excel worksheets.

  • Integration with Worksheet Class: We have seamlessly integrated the ValidationRule application within the Worksheet class. This integration allows for dynamic and flexible data validation across different cells and scenarios, making it easier to ensure data accuracy.

  • Enhanced Range Class: The Range class has been adapted to apply ValidationRules to specific cell ranges. This enhancement enhances the granularity and control of data validation in your worksheets, giving you more precise control over data validation rules.

  • Robust Unit Tests: To ensure the reliability and correctness of these new features, we have crafted and added a suite of robust unit tests for each type of ValidationRule. These tests cover the application of ValidationRules in various scenarios, validating their functionality and effectiveness in maintaining data integrity.

  • Compatibility and Quality Assurance: We have thoroughly ensured that the implementation is compatible with existing functionalities and adheres to project standards for code quality and performance. You can trust these new features to seamlessly integrate with your existing codebase.

Overall, this update significantly bolsters the data handling capabilities of our Excel processing toolkit. It provides advanced options for enforcing data integrity and guiding user inputs within worksheets, making FileFormat.Cells even more powerful and user-centric.

Upgrade to the latest version of FileFormat.Cells to take advantage of these dynamic data validation features and enhance your Excel automation workflows.
