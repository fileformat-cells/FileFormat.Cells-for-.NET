FileFormat.Cells for .NET

This is the homepage of FileFormat.Cells for .Net which is an open-source .NET API to automate Excel documents. It is an enterprise-level C# API built on top of OpenXML that Microsoft backs. The installation process is quite easy and the developers can use its features as per their business requirements.

Further, this Spreadsheet automation API is extensible and comes with comprehensive documentation regarding installation and usage.

FileFormat.Cells for .Net is lightweight and available as a Nuget Package. To install this .NET API, just run the command in the NuGet Package Manager:

Install-Package FileFormat.Cells

Once it is installed, you are all set to start creating and manipulating Spreadsheets programmatically. Moreover, the source code of this open-source API is available on the Github repository. Therefore, please stay in touch for the upcoming features.
